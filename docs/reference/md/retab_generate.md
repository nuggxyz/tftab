## retab generate



```
retab generate [flags]
```

### Options

```
  -d, --debug         Print debug output
      --file string   location of the retab.hcl file (default "retab.hcl")
  -h, --help          help for generate
  -q, --quiet         Do not print any output
  -v, --version       Print version and exit
```
