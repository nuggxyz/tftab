## retab lsp

run a server for retab code using the Language Server Protocol

```
retab lsp [flags]
```

### Options

```
  -d, --debug     Print debug output
  -h, --help      help for lsp
  -q, --quiet     Do not print any output
  -v, --version   Print version and exit
```
