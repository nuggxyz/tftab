## retab 

install buildrc

```
retab  [flags]
```

### Options

```
  -d, --debug     Print debug output
  -h, --help      help for this command
  -l, --latest    Install the latest version
  -q, --quiet     Do not print any output
  -v, --version   Print version and exit
```
